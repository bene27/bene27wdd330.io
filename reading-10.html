MDN: Using Fetch()

  1. The fetch API provides a JavaScript interface for accessing and manipulating parts of the HTTP pipeline
  2. Fetch differs from jQuery.ajax() in three main ways
  3. Fetch method accepts a second parameter
  4. To cause browsers to send a request with credentials include same origin and cross origin calls
  5. Use fetch() to POST JSON encoded data
  6. File upload and formData() and fetch()
  7. Checking that the promise is resolved then checking that the response.ok property has a value of true.This checks if the fetch was successful
  8. Fetch API support can be detected by checking for the existence of Headers, Reqest, Response or Fetch

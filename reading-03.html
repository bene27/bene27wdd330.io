Events

  • Event listeners are like setting a notification to alert you when something happens
  • Event listners are part of DOM and they are added elements on a page.
  • The click event takes place when a user clicks on the mouse or screen
  • When a event is invoked, it callsback a function
  • The event target returns a reference to the node
  • Be careful of including both the click and double click event in the same event
  • Mousemove is when the mouse moves
  • Keydown takes place when a key is pressed
  • Keyup takes place when a key is released
  • Modifiers have a purpose already in the browser
  • Touchstart happens when the user touches the surface
  • A event listner can be removed
  • Delegation is the process of assigning an event listner to a parent element

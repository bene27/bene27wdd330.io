Computer programming basics
  • Javascript has seven data types
  • Variables are used in programming languages to refer to a value stored in memory
  • Variables have to be declared and then assigned
  • You can declare and assign multiple variables at the same time
  • Any variable declared outside a block is said to have global scope.Meaning it is accessible everywhere in the program
  • Backslahes are used to escape special characters in strings
  • A method is an action that primitive data type can perform.We use the dot operator followed by the name of the operator
    • Examples:
    .toUpperCase
    .indexOf
    .includes
  • The trim() method moves whitespaces
  • Symbols are used to create unique values which help avoid any name collisions
  • Hex numbers are used for clor codes
  • Infinity is a special error value used to represent large numbers
  • Nan is an error value short for 'not a number'
  • parseint converts string to integar
  • parsefloat converts string to float

Arrays, Logics and Loops
  • Arrays is an ordered list of values
  • First values in an index is indicated with a 0
  • Destructuring is taking values out of an array and presenting its indiviual values
  • Length-1 can be used to find the last item of a list and also length alone can be used to find the length of an array
  • You can also merge arrays by using concat()
  • Multidimensional arrays is used to create a coordinate system
  • Loops repeat a piece of code
  • Maps are lists of key value pairs
  • Sets are lists of non duplicate values
  • If and else statements control the flow of a code.

Functions
  • To write a function do the following function functionName(ags,ags){}
  • To run a code simply enter its name, followed by parentheses
  • Arguments are values that are provided to a function when it is invoked
  • The rest operator is used to access multiply arguments as an array.

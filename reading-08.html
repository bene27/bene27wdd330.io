Chapter 8:Transforms and Transitions

  1. The CSS3 tansform property enables one to translate, rotate, scale and/or skew any elements on the page.
  2. You can manipulate an element appearance using transform function
  3. Translation moves elements around
  4. Transforms don't work on inline elements
  5. The scale function scales an element by the defined factors horizontally then vertically
  6. A scaled element will grow outwards from or shrink inwards towards its center
  7. Scaling and translation has no impact on the document flow
  8. The rotate function rotates an element around the point of origin by specified angle value
  9. A skew function specifies a skew along the x and y axes
  10. The order of transform function does matter
  11. Native CSS transitions generally require less client side processing
  12. Transitions allow the value of CSS properties to change over time
  13. CSS transitions are delclared along with the regular styles on an element
  14. Transitions durations property sets how long the transitions will take
  15. Transitions timing functions lets one control the pace of the transition in more granular detail
  16. Transition properties allow for multiple transitions in one call
  17. CSS animations allow you to control each step of animation in more complex ways
  18. Animation iteration count property defines how many times the animation will play through
